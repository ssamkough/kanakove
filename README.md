# KanaKove 🏯

A website to help people learn the Japanese Kana (hiragana, katakana).

## Technologies 🎇

-   [React](https://reactjs.org/)
-   [Typescript](https://www.typescriptlang.org/index.html)
-   [Ant Design](http://ant.design/)
